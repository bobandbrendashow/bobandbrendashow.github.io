TikTok Uploader

TikTok Uploader is a lightweight application that allows users to upload and publish videos to their TikTok account after explicitly authorising access.

What this app does

  • Authenticates users via TikTok Login (OAuth)
  • Uploads MP4 videos selected by the user
  • Publishes content using TikTok’s official Content Posting API

User control & transparency

The app only performs actions that the user explicitly requests. Access can be revoked at any time from TikTok’s account settings.
